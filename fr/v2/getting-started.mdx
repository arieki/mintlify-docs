## Configuration des fonctionnalités IA

### Configuration de prévision de demande

Activez la prévision de demande alimentée par IA pour prédire les besoins en salsa 7-30 jours à l'avance :

<Steps>
   <Step title="Activer les prévisions">
      Allez dans **Fonctionnalités IA > Prévision de demande** et activez la fonctionnalité.
   </Step>
   <Step title="Connecter les sources de données">
      Liez votre système POS, votre système de réservation et toutes sources externes (API météo, calendriers d’événements locaux).
   </Step>
   <Step title="Importer les données historiques">
      Importez au moins 90 jours de données de ventes historiques pour une précision optimale. L’IA s’améliore avec le temps.
   </Step>
   <Step title="Définir les paramètres">
      Configurez l’horizon de prévision (7, 14 ou 30 jours), les intervalles de confiance et les seuils d’alerte.
   </Step>
   <Step title="Revoir la première prévision">
      L’IA génère votre première prévision sous 24 heures. Revoyez et ajustez si nécessaire.
   </Step>
</Steps>

<Note>
   **Précision des prévisions** : L'IA atteint une précision de 70-80% avec 90 jours de données et s'améliore à 95%+ après 6 mois.
</Note>

### Prédiction de qualité

Configurez la surveillance qualité par IA pour détecter les problèmes avant qu’ils ne surviennent :

<CodeGroup>
```javascript JavaScript
// Exemple : configurer la surveillance qualité via l’API
const qualityConfig = {
  enabled: true,
  monitoring_interval: "5m",  // Vérification toutes les 5 minutes
  thresholds: {
    temperature: { min: 35, max: 40, unit: "F" },
    freshness_score: { min: 8.0, max: 10.0 },
    consistency: { min: 7.5 }
  },
  alert_channels: ["email", "sms", "push", "slack"],
  auto_action: {
    low_quality_detected: "pause_production",
    temperature_violation: "notify_manager"
  }
};
```
</CodeGroup>

### Intégration ERP

Connectez v2.0 à votre système ERP :

<CodeGroup>
```python python
# Exemple : configuration d’intégration ERP
from salsa_sdk import SalsaClient

client = SalsaClient(api_key="your_api_key")

# Connexion à SAP
erp_connection = client.integrations.connect_erp(
    system="sap",
    credentials={
        "host": "erp.yourcompany.com",
        "client": "100",
        "username": "SALSA_INT",
        "auth_type": "oauth"
    },
    sync_options={
        "inventory": True,
        "procurement": True,
        "cost_accounting": True,
        "frequency": "real-time"
    }
)

print(f"ERP connected: {erp_connection.status}")
```
</CodeGroup>
